A little bit more about me

I’ve been working as a Search Engine Optimiser for almost a decade and fell into it when I took an interest in blogging as a student. I have a natural interest for all things organised and pretty and taught myself basic HTML and CSS to design web pages and then found out about Search algorithms. That’s literally how I got into the industry!

If you’re not too familiar with SEO, it is kind of at the crossroads of marketing and web development and understanding basic code is part of the job. You don’t have to be an expert coder to be a good SEO, but coding expertly can make you a great SEO.

My goal this year is to not only upskill by learning Python, Javascript, Cloud development, Django and React, but also to transition from a Marketing/SEO role to a Web development/SEO role.

So here I am today, taking the leap and learning as much as I can to become a full stack developer :)
